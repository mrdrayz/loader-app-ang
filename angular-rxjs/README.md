# RxJS Image Downloader

Приложение для параллельного скачивания изображений с использованием **RxJS**  
и сохранения их на диск через **File System API** (только для Chrome).

Из-за использования экспериментального File System API приложение работает только в Google Chrome.

## Функциональность

- Загрузка изображений запускается по кнопке **Download images**
- Перед загрузкой пользователь выбирает директорию (через `showDirectoryPicker()`)
- Одновременно скачивается не более **3 изображений**
- Остальные изображения ожидают своей очереди
- Ошибка загрузки одного изображения не прерывает процесс
- Кнопка блокируется на время загрузки
- Для каждого изображения выводится сообщение в консоль:
    - успешно сохранено
    - ошибка
- После завершения отображается итоговая статистика:
    - общее количество
    - успешные
    - ошибки

## Технологии

- **TypeScript**
- **RxJS 7**
- **Vite**
- **File System Access API**
- **DOM API**

## Особенности реализации:

- Используется только **один вызов `subscribe()`**
- Ограничение параллельности реализовано через `mergeMap(..., 3)`
- Все вычисления статистики выполняются внутри RxJS-пайплайна (`reduce`)
- Ошибки отдельных загрузок обрабатываются через `catchError`

## Установка и запуск

```bash
npm install - Установка зависимостей
```

```bash
npm run dev - Запуск проекта в режиме разработки
```

После запуска приложение будет доступно по адресу: http://localhost:5173/

## Настройка параллельности

Количество одновременно загружаемых изображений можно изменить в файле downloadImage.ts в строке const MAX_CONCURRENT_DOWNLOADS = 3;
